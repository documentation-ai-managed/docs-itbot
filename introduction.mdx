## Overview

ITBot empowers teams to centralize project documentation in a collaborative space. You create, edit, and version docs effortlessly, with features like real-time editing, powerful search, and integrations. Streamline your workflow by replacing scattered wikis, Notion pages, and Google Docs. Benefits include reduced onboarding time for new hires, consistent knowledge sharing, and easy maintenance as projects evolve.

## Quick Start

Get up and running in under 5 minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com` and create a free account. No credit card required.
  </Step>
  <Step title="Create Your First Space" icon="folder">
    Name it for your project, like "MyApp Docs". Invite team members via email.
  </Step>
  <Step title="Add Content" icon="plus">
    Create pages using Markdown or our visual editor. Here's a simple API integration example:
    
    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const response = await fetch('https://api.example.com/docs', {
        method: 'POST',
        headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
        body: JSON.stringify({ title: 'New Doc', content: '# Hello ITBot' })
      });
      const doc = await response.json();
      console.log(doc.id);
      ```
      ```python
      import requests
      
      response = requests.post(
        'https://api.example.com/docs',
        headers={'Authorization': f'Bearer {YOUR_API_KEY}'},
        json={'title': 'New Doc', 'content': '# Hello ITBot'}
      )
      doc = response.json()
      print(doc['id'])
      ```
    </CodeGroup>
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Set permissions and generate public links. Embed in your apps or sites.
  </Step>
</Steps>
