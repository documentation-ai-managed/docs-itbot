## Prerequisites

Before you begin, ensure you have:

- A valid email address for account verification
- Access to a modern web browser like Chrome or Firefox

<Callout kind="info">

ITBot requires email verification to enable collaboration features. Check your spam folder if the verification email does not arrive within 5 minutes.

</Callout>

## Create Your Account

Follow these steps to sign up quickly.

<Steps>
  <Step title="Visit the Signup Page" icon="globe">
    Navigate to `https://app.itbot.com/signup` in your browser.
  </Step>
  <Step title="Enter Your Details">
    Provide your email, create a strong password, and complete the CAPTCHA.
  </Step>
  <Step title="Verify Your Email">
    Check your inbox for the verification link from ITBot and click it to activate your account.
  </Step>
</Steps>

## Set Up Your First Documentation Space

Once logged in, create a space to organize your project docs.

<Steps>
  <Step title="Create New Space" icon="plus">
    Click the "New Space" button in the dashboard.
  </Step>
  <Step title="Configure Space">
    Enter a name like "Project Alpha Docs", select visibility (private or public), and choose a brand color such as `#3B82F6`.
  </Step>
  <Step title="Add Initial Pages">
    Create pages for "Overview", "API Reference", and "Changelog" using the page creator.
  </Step>
</Steps>

## Invite Team Members

Collaborate by adding your team.

<Tabs>
  <Tab title="Email Invite" icon="mail">
    In your space settings, enter team emails and send invites. They receive a link to join.
  </Tab>
  <Tab title="Share Link" icon="link">
    Generate a shareable link from the space menu and distribute it via Slack or email.
  </Tab>
</Tabs>

<Callout kind="tip">

Set role permissions during invites: Admin for full access, Editor for content changes, Viewer for read-only.

</Callout>

## Make Your First Edit

Edit a page and add a code example.

<Steps>
  <Step title="Open a Page" icon="file-text">
    Select "Overview" from your space navigation.
  </Step>
  <Step title="Edit Content">
    Click "Edit" and add content using the toolbar.
  </Step>
  <Step title="Add API Example">
    Insert a code block for an API call to fetch space data.
  </Step>
  <Step title="Publish Changes">
    Click "Publish" to make edits live.
  </Step>
</Steps>

Here is a sample API request to list your spaces:

<CodeGroup tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/spaces', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    }
  });
  const spaces = await response.json();
  console.log(spaces);
  ```
  ```bash
  curl -X GET https://api.example.com/v1/spaces \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json"
  ```
</CodeGroup>
